## Sääpäiväkirja

Käyttäjä voi kirjata sovellukseen päivittäiset säähavaintonsa, ja tarkastella aiemmin kirjaamiaan havaintoja. Sovellukseen voi luoda käyttäjän, joka yksilöi omat säähavainnot.
